# Game App Example

This project uses cmake as a build system, SDL3 as a game library, and C23 as a programming language.

## Prerequisites

- CMake (version 3.30 or higher)
- SDL3 (version 2.28 or higher)
- C23 compiler

## Building

```bash
mkdir build
cd build

# On Linux
cmake ..
make

# On Windows
# cmake ..
# cmake --build .
```
